# 🤖 AI Voice Assistant Chatbot

## 📌 Overview

AI Voice Assistant Chatbot is a portable AI-powered chatbot designed to be integrated into websites using a simple script. The project focuses on enabling intelligent real-time customer interaction through voice communication, multilingual support, emotion detection, and simulated business integrations.

The chatbot is designed to provide a scalable and interactive AI experience for modern web applications.

---

# ✨ Features

- 🎙️ Voice input and voice response
- 🤖 Natural conversational AI
- 😊 Emotion detection
- 🌍 Language detection and multilingual interaction
- 🛡️ Fraud detection (simulated)
- 📊 CRM integration with mock data
- 📦 ERP inventory checking system
- 💬 Portable chatbot widget
- 🌐 Website integration support
