# Lensora 👓🤖
AI Powered Conversational Commerce Platform

## 📌 Overview

Lensora is an AI-powered conversational commerce platform developed during a hackathon to demonstrate how intelligent virtual assistants can transform online shopping experiences.

Instead of building just another ecommerce website, the core idea behind Lensora was to create a multilingual AI assistant capable of interacting with users naturally through both text and voice while being integrated into a realistic shopping environment.

The platform combines ecommerce functionality with conversational AI, voice interaction, multilingual support, and real-time order assistance.

# 🎯 Hackathon Vision
The primary goal of Lensora was to demonstrate how conversational AI and voice interaction can improve ecommerce experiences.

Rather than building only a chatbot, the project focused on creating a realistic commerce ecosystem where AI could:

* interact naturally,
* understand multilingual queries,
* assist with orders,
* provide recommendations,
* and improve customer engagement.
